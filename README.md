# Fake News Detection using BERT

## Problem Statement
Fake news spreads rapidly on the internet and can mislead people. The objective of this project is to classify news articles as Fake or Real using NLP.

## Approach
Loaded dataset, combined text, tokenized using BERT tokenizer, split data, fine-tuned BERT model, and evaluated performance.

## Model Used
bert-base-uncased

## Metrics
Accuracy, Precision, Recall, F1-score, Confusion Matrix

## Improvements
Tried improving performance by tuning parameters and handling data imbalance.

## Key Learnings
Learned tokenization, transformer models, fine-tuning, and evaluation of NLP models.
